# Mindfine

Sistema web voltado ao apoio à saúde mental e bem-estar emocional, desenvolvido como Projeto Integrador.

## Sobre o Projeto

O Mindfine é uma plataforma digital que oferece ferramentas lúdicas e terapêuticas para crianças e jovens. O sistema busca promover o autoconhecimento emocional por meio de recursos interativos, contribuindo para o desenvolvimento saudável dos usuários.

## Funcionalidades

- Ilhas temáticas com ambientes interativos
- Sala de jogos (memória, jogo da velha, caça-palavras, desenho)
- Diário emocional
- Calendário de emoções
- Playlist com sons relaxantes
- Página de respiração guiada
- Chat com psicóloga
- Área escolar
- Personagem customizável

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## Estrutura do Projeto

O repositório contém as páginas HTML, arquivos de mídia (imagens e sons), scripts JavaScript e demais artefatos desenvolvidos até o momento.
